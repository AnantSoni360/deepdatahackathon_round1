## 🚀 Quick Start

### Prerequisites
1. **Python 3.7+** installed on your system
2. **Web browser** (Chrome, Firefox, Safari, Edge)
3. **Terminal/Command Prompt** access

### Installation & Setup
```bash
# 1. Navigate to the folder
cd final_deepdatahack

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the dashboard
streamlit run esg_dashboard_final.py
```

### 🌐 Dashboard Access
After running the command, you'll see:
```
Local URL: http://localhost:8524
Network URL: http://192.168.1.100:8524

Ready to go! 🚀
```
**Click on the Local URL or copy-paste it into your browser**

## 🆘 Troubleshooting

### Common Issues & Solutions

#### ❌ Port Already in Use
```
Error: Port 8524 is already in use
Solution: streamlit run esg_dashboard_final.py --server.port 8525
```

#### ❌ Module Not Found
```
Error: ModuleNotFoundError: No module named 'streamlit'
Solution: pip install -r requirements.txt
```

#### ❌ Dataset Not Found
```
Error: FileNotFoundError: company_esg_financial_dataset.csv
Solution: Ensure all files are in the same folder
```
